# Retail Performance Hub: Optimizing Low-Margin Product Portfolios

## Project Overview
This project analyzes retail sales data to identify factors affecting profitability and support data-driven business decisions.

## Business Problem
The company has high sales but shrinking profit margins due to discounts, returns, and regional performance differences. This project identifies key factors affecting profitability.

## Objectives
- Analyze sales and profit trends
- Study the impact of discounts on profit
- Perform regional sales analysis
- Identify top-performing customers and products
- Build an interactive Power BI dashboard
- Generate business insights using SQL, Python, and Statistics
